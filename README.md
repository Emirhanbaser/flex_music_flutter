### Kurulum Adımları

1. Projeyi bilgisayarınıza indirin:

```bash

git  clone [https://github.com/Emirhanbaser/flex_music_flutter.git]

```

  

2. Proje klasörüne gidin:

```bash

cd [flex_music_flutter]

```

  

3. Gerekli paketleri yükleyin:

```bash

flutter  pub  get

```

  

4. Uygulamayı başlatın:

```bash

flutter  run

```
